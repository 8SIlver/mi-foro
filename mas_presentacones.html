SCESI - Git & GitHub

Git es un sistema de control de versiones que permite llevar el historial de los cambios en tu código.

GitHub es una plataforma que aloja tus repositorios Git en la nube y permite colaborar fácilmente con otros desarrolladores.

Con Git puedes clonar, hacer commits, crear ramas y fusionar cambios. Con GitHub puedes publicar proyectos, abrir pull requests y mostrar tu trabajo al mundo.
